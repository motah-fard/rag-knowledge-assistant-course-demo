Lab 2 - Build a Retrieval Engine

Objective

Build a semantic search engine using embeddings.

⸻

Requirements

Step 1

Generate embeddings for every chunk.

⸻

Step 2

Accept a user question.

Example:

How many PTO days do employees get?

⸻

Step 3

Generate an embedding for the question.

⸻

Step 4

Calculate cosine similarity.

⸻

Step 5

Return the top 3 chunks.

⸻

Test Questions

Question 1:

How many vacation days do employees get?

Question 2:

What hotel budget is allowed?

Question 3:

Can employees share passwords?

⸻

Expected Result

The correct policy document should appear in the top retrieved results.

⸻

Challenge

Compare:

* keyword search
* embedding search

Questions:

1. Which performs better?
2. Why?
3. When does keyword search fail?
