# Assignment 1 - Prompt Engineering

Goal:
Improve the assistant's behavior.

Task:
Modify the prompt so the assistant:

- Does not guess
- Answers briefly
- Says "I do not know" when information is missing

Test Questions:

1. How many PTO days do employees get?
2. What is the company bonus policy?
3. Can employees work remotely from another country?

Expected Result:

The assistant should avoid hallucinating answers.
